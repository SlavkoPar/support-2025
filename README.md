## MERN Stack
**MERN** stands for MongoDB, Express, React, Node, after the four key technologies that make up the stack.\
**M**ongoDB — document database.\
**E**xpress(.js) — web framework.\
**N**ode.js\
**R**eact(.js) — a client-side JavaScript framework.\
